## 🎮 About The Game

**PactumX** is a fast-paced action roguelike built in Unity, centered around a unique mechanic:

> ⚖️ **Power comes with a price.**

Players gain powerful abilities through **pacts**, but each one introduces a permanent drawback.  
The more power you seek, the more unstable your run becomes.

Every decision matters.

---

## ⚙️ Core Features

### 🧠 Pact System (Unique Mechanic)
- Gain buffs with permanent curses
- Stackable effects that reshape gameplay
- Emergent builds every run

### ⚔️ Skill-Based Combat
- Responsive movement and attacks
- Focus on positioning and timing
- High-risk, high-reward gameplay

### 🌀 Procedural Generation
- Dynamic room layouts
- Replayable runs with variation

### 👁️ Adaptive Final Boss
- Boss evolves based on your build and choices
- Reflects your pact decisions

### 🌑 Dark Fantasy / Lovecraftian World
- Cosmic horror themes
- Distorted entities and forbidden power
